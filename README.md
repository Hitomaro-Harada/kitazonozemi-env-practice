# kitazonozemi-env-practice

北園ゼミの **Git / GitHub / Docker を用いた開発環境構築の練習用リポジトリ** です。

## 手順
1. 新しいリポジトリを作ってみよう
2. 作ったリポジトリをローカルに clone しよう
3. README を書き換えて `git add` `git commit` `git push` してみよう
   - Git と GitHub がきちんと接続されているかを確認する
4. ディレクトリを整えよう
5. Docker に接続するための準備をしよう
6. Docker に接続してみよう
7. Python プログラムを実行してみよう
8. GitHub でもう一度リポジトリを見てみよう

## 余裕があれば
- Codex を使ってコーディングしてみよう
